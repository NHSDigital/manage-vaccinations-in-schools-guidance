## Before you start

Read an overview of [what records you need to upload to Mavis and when](/what-to-upload-and-when/). 

## File requirements

Before uploading a file, check:

- your records are in the format shown in our templates 
- you’ve included all information marked as ‘Required’ in our templates 
- your file is saved in CSV (.csv) format
- your file contains no more than 10,000 rows (split larger files into smaller files before uploading)

If you have an Excel file with multiple tabs, before uploading it check you have either:

- combined the tabs into a single worksheet before saving it as a CSV file, or 
- created a separate CSV file for each tab
